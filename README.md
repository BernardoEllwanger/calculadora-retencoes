# Calculadora IRRF + ISSQN

Calculadora de retenções de impostos para autônomos (RPA — Recibo de Pagamento a Autônomo).

## O que calcula

Dado o valor bruto de uma nota:

- **ISSQN**: alíquota fixa de 4%
- **IRRF**: tabela progressiva com isenção até R$ 5.000,00, alíquota de 27,5% e redutor da nova lei (MP 1.206/2024)
- **Valor líquido**: valor bruto descontadas as retenções

## Stack

HTML · CSS · JavaScript

## Como rodar

Abra `index.html` no navegador. Sem dependências, sem build.
